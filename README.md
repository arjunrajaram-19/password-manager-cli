# Password Manager (CLI)
A command-line based password manager built using Python.

## Features
- Add account (website, username, password)
- View saved accounts (passwords masked)
- Search account by website
- Delete account
- Show password with confirmation
- Persistent storage using file handling

## Tech Used
- Python
- File Handling

## How to Run
1. Download or clone the repository
2. Run main.py
